■説明
  このスクリプトは、サイト「ゆうぼくみん」用のIRCボットです。
  
■導入方法
  LimeChatをインストールした後に、以下の手順で導入してください。
  
  (1) LimeChat を管理者として実行する
  (2) メニューの[設定]を選ぶ
  (3) [スクリプトの設定]を選ぶ
  (4) ボタンの[スクリプトフォルダを開く]を押す(すると、フォルダが開く)
  (5) JScriptファイル(「***.js」となっているファイル)をコピーする
  (6) JScriptファイルを開き、「var targetChannel = '#xxx';」と書いてある行の#xxxの部分をBOTを”実行したいチャンネル名”に書き換え、保存する。例.「var targetChannel = #hoge」。
  (7) LimeChatのウィンドウを再びアクティブにさせる。
  (8) メニューの[設定]を選ぶ
  (9) [スクリプトの設定]を選ぶ
  (10) サーバーが登録されているなら、ファイル名のすぐ右側に、各サーバーの列が追加されているので、右クリックし、「〇」を付ける
  (11) [スクリプトの設定]を閉じる
  
■著作権について
  原則、コピーフリーです。
  改変も自由にしていいですが、勝手に正規品と偽ったり、サイトに負荷をかけたり、
  悪質だと作者が判断した場合は、「改変しないままの利用」やコピー・改変も許可しません。
